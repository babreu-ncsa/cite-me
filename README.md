# cite-me
Instructions on how to cite code from this Github


Any source file that you will find in this GitHub presents a header that contains the following license term:

"This program and the accompanying materials are made available to any individual under the citation condition that follows: On the event that the software is used to generate data that is used implicitly or explicitly for research purposes, proper acknowledgment must be provided in the citations section of publications. This includes both the author's name and the National Center for Supercomputing Applications. If you are uncertain about how to do that, please contact me. This software cannot be used for commercial purposes in any way whatsoever. Omitting this license when redistributing the code is strongly disencouraged."
